# **App Name**: ChainCanvas

## Core Features:

- WalletConnect Integration: Integrate WalletConnect to support multiple wallets like MetaMask, Coinbase Wallet, and Rainbow, displaying connected wallet address and Base balance with disconnect functionality.
- Multi-Chain NFT Portfolio Viewer: Fetch and display NFTs from Base, Ethereum, Polygon, and Optimism using Alchemy API, presenting them in a grid layout with image, name, collection, and chain badge, including filters and search functionality.
- NFT Marketplace: List NFTs for sale with prices in ETH/USDC, browse listed NFTs with sorting options, and facilitate NFT purchases using Base as the settlement layer via a smart contract for escrow and automated transfers, showing transaction status.
- User Profiles: Enable user profiles displaying owned NFTs and activity, with functionality to follow/unfollow collectors, fostering a social environment within the platform.
- NFT Detail Page: Add comment sections on each NFT detail page, plus liking/favoriting NFT functionality. Show an activity feed that shows purchases, listings, and follows.
- Gas Optimization Tool: Show estimated gas costs before transactions; a tool will suggest to the user when to perform batch transfers of multiple NFTs for the sake of gas optimization, leveraging Base's L2 benefits.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) for a sophisticated and modern look, reflecting the advanced blockchain technology.
- Background color: Dark gray (#222222), subtly desaturated to create a sleek dark mode interface.
- Accent color: Teal (#008080), providing a contrasting highlight for interactive elements and notifications.
- Body text: 'Inter' sans-serif for a modern, neutral look; Headline font: 'Space Grotesk' sans-serif, suitable for headlines
- Use minimalist, outlined icons to represent different chains and functionalities within the marketplace.
- Implement a responsive grid layout that adapts to different screen sizes, ensuring optimal viewing experience on all devices.
- Incorporate smooth transitions and subtle animations to enhance user engagement and provide visual feedback.
